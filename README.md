# Dotfiles

Collection of config files from my notebook Accer Aspire running Debian 13. 

Includes:
    - Bash
    - Zshrc
    - Aliases
    - NeoVim (config & plugins)
    - Midnight Commander
    - Micro
    - Xfce config

## How to replicate my setup

1. Clone repo.
2. Install GNU Stow.
3. Use `stwo .` inside the folder to create symbolic links.
